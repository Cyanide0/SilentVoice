التقنيات المستخدمة Technologies Used

استعراض مفصّل للمكتبات والأطر والبنية التحتية التي يقوم عليها النظام، مقسّمة إلى أقسام واضحة. A detailed look at the libraries, frameworks, and infrastructure the system is built on, organized into clear sections.

لغة البرمجة والمكتبات الأساسيةProgramming Language & Core Libraries
Python
لغة البرمجة الرئيسية للمشروع بأكمله، تتميّز بنظام بيئي غني لتطبيقات الذكاء الاصطناعي ورؤية الحاسوب.The primary programming language for the entire project, featuring a rich ecosystem for AI and computer vision applications.
v3.10+
OpenCV
مكتبة رؤية الحاسوب الأساسية لالتقاط الفيديو، معالجة الإطارات، التطبيع، وعزل مناطق اليد.The core computer vision library for video capture, frame processing, normalization, and hand region isolation.
cv2
MediaPipe
إطار عمل من Google لتتبّع نقاط اليد الـ٢١ بدقة عالية في الوقت الفعلي مع دعم ثلاثي الأبعاد.Google's framework for tracking 21 hand landmarks with high accuracy in real time, including 3D support.
Hands API
PyTorch
الإطار الرئيسي لبناء وتدريب نماذج التعلّم العميق (CNN + LSTM) مع دعم GPU متكامل.The primary framework for building and training deep learning models (CNN + LSTM) with full GPU support.
v2.0+
TorchVision
مكمّل PyTorch للتعامل مع مجموعات البيانات البصرية، التحويلات، والنماذج المدرَّبة مسبقاً.PyTorch's companion library for visual datasets, transforms, and pre-trained models.
Transforms
NumPy + Pandas
العمليات العددية، معالجة المصفوفات، إدارة مجموعات البيانات، وترميز التسميات.Numerical operations, array processing, dataset management, and label encoding.
Array + DataFrame

TTS

Cognitive Services

استراتيجية النماذجModel Strategy
CNN — التعلّم المكانيCNN — Spatial Learning
تستخرج الشبكات التلافيفية السمات المكانية من إطارات الفيديو أو صور نقاط المعالم.Convolutional networks extract spatial features from video frames or landmark images.
Spatial Features
LSTM — التسلسل الزمنيLSTM — Temporal Sequence
تُحلّل LSTM التتابع الزمني للإشارة عبر الإطارات المتعاقبة لفهم الإشارات الديناميكية.LSTM analyzes the temporal sequence of signs across consecutive frames to understand dynamic gestures.
Temporal Sequences
Transfer Learning
استخدام نماذج مدرَّبة مسبقاً ثم الضبط الدقيق لتخصيصها لمجموعة بيانات لغة الإشارة المحدودة.Using pre-trained models then fine-tuning them for the limited sign language dataset.
Fine-tuning
YOLO — كشف الإشارةYOLO — Sign Detection
يُستخدم YOLO للكشف السريع عن الإشارة وتحديد موضعها في الإطار كخطوة أولى.YOLO is used for rapid sign detection and localization within the frame as a first step.
Detection

+2000 كلمة+2000 words

29 حرف29 letters
